Word Scramble

Flip a card. Unscramble the word. Shout it before they do.

Free · No ads · No accounts · A room is required to play

How to Play

  1. Host a Game — pick your categories, team names, whether Letter Hints are on, an optional Round Timer, and an optional Target Score. Tap Start Room to get a 4-letter room code.
  2. Join as Display — on the TV or laptop everyone can see, open this same page and enter the Host's room code.
  3. Start Game — the Host taps Start once the Display is connected.
  4. Each card shows a word's letters scrambled out of order. A row of blank letter tiles shows the word's length underneath.
  5. Shout it out — no typing, no tracing, no buzzers. Teams just call out the unscrambled word as soon as they think they've cracked it.
  6. The Host taps Team A got it / Team B got it to award the point and deal the next card, or Skip if nobody can get it.
  7. If Letter Hints are on, the Host can Reveal a Letter at any time.
  8. If a Round Timer is set, the Host taps Start Timer once, right when the round is ready to go — it's one clock for the whole round, not per card. Award and skip through as many cards as you can before it hits zero; when it does, the round ends immediately and the higher score wins.
  9. If a Target Score is set, the first team to reach it wins instantly. Otherwise, the game ends when the deck runs out — highest score wins (a tie is a draw).

A working room is required to play: the Host screen shows the answer, so it can't double as the shared Display.
